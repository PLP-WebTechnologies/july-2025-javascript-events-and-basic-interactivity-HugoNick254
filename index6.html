FAQ

What is JavaScript used for? +

JavaScript is a programming language that enables you to create dynamically updating content, control multimedia, animate images and much more. It's used on almost every website.

How do I add an event listener? +

You use the `.addEvetListener()` method on an element. It takes 2 arguments: the type of event to listen for (e.g. `'click'`), and the function to run when the event occurs.

What is the DOM? +

The Document Object Model (DOM) is a porgramming interface for web documents. It represents the page so that programs can change the document structure, style and content.
